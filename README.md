# Pfun

Pfun (Parse, FUzz, and Normalize) watches a directory containing JSON files, parses the files, anonymizes the data, and normalizes the data in a database.

## Getting Started

### Prerequisites

Use the package manager Homebrew to install gradle.

``` bash
$ brew install gradle
```

### Building

``` bash
$ ./gradlew build
```

### Running

``` bash
$ ./gradlew run --args="pathToDirectoryToWatch"
```
